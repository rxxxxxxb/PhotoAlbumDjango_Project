# PhotoAlbumDjango_Project
Django photo album project

#This is a photo album project with django

#Description:

User has 2 role:
1. Viewer 
2. Registered  User


Viewer Actions:
- Can view the album
- Can view the photo
- Can Sign Up
- Can Log in



Registered User Actions:
- Can create album
- Can Post photo individually and in an specific album
- Can Like and Unlike an album
- Can delete photo 
- Can view the album
- Can view the photo
- Can Log in
